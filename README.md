# 分段拟合算法文档

## 1. 项目概述

本项目实现了一种基于自适应阈值的迭代式分段拟合算法，用于拟合ECG（心电图）数据曲线。该算法能够智能识别数据中的水平直线段和直线段，通过迭代式的分段点识别机制，实现高精度的曲线拟合。

### 1.1 算法特点

- **智能分段**：自动识别水平直线段和直线段，减少分段数量
- **自适应阈值**：从100开始逐步调整阈值，确保拟合精度
- **高精度拟合**：对不同类型的线段采用不同的拟合策略
- **鲁棒性强**：能够处理各种复杂的ECG数据曲线

## 2. 数据说明

- 第一列：时间x

## 3. 算法实现

### 3.1 核心算法流程

1. **数据加载**：读取ECG数据文件
2. **分段点识别**：
   - 水平直线段检测：通过计算连续5个点的y值标准差识别水平直线
   - 直线段检测：通过斜率变化阈值识别直线段
   - 自适应阈值分段：从阈值100开始，逐步缩小10倍，寻找最佳分段点
3. **分段拟合**：对每个分段使用多项式或线性拟合
4. **结果评估**：计算均方误差（MSE）评估拟合精度

### 3.2 分段点识别机制

分段点识别是算法的核心，采用以下策略：

1. **水平直线检测**：
   - 计算连续5个点的y值标准差
   - 如果标准差小于1e-8，认为是水平直线
   - 扩展找到水平直线的结束点

2. **直线段检测**：
   - 计算起始点与第二个点的斜率作为基准斜率
   - 检查后续点的斜率变化
   - 如果斜率变化小于0.000001，尝试直线拟合
   - 如果MSE小于0.000001，认为是直线段

3. **自适应阈值分段**：
   - 阈值初始值设为100
   - 计算起始点与各点的斜率差
   - 当斜率差大于阈值时，计算该分段的MSE
   - 如果MSE大于0.000001，缩小阈值10倍重新寻找

## 5. 算法优化

### 5.1 水平直线段异常分析

**问题**：水平直线段（如900-999点）每个点被识别为分段点，导致分段数量过多。

**原因**：

- 浮点精度误差导致微小的斜率变化
- 算法对水平直线的判断不够严格

**解决方案**：

- 添加水平直线检测逻辑，通过计算连续5个点的y值标准差识别水平直线
- 对水平直线进行批量处理，作为一个完整的分段

### 5.2 斜率计算优化

**优化前**：使用"起始点与分段点的斜率"判断
**优化后**：使用"起始点与第二个点的斜率减去起始点与分段点的斜率的绝对值"判断

**优势**：

- 更准确地反映斜率变化
- 对微小变化不敏感
- 提高分段的合理性

### 5.3 直线段检测

**新增功能**：

- 检测斜率变化小于0.000001的区域
- 尝试直线拟合，保留MSE小于0.000001的直线段
- 减少不必要的分段，提高拟合效率
